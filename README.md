# h5-cn-resource

本仓库用于归档和发布多个独立的 HTML 页面资源。这些页面通常为单文件或少量资源文件组成，不依赖于特定域名或具体网站。

## 项目简介

随着前端技术的发展，许多轻量级 HTML 页面具有独立的实用价值。本仓库旨在收集、整理并长期保存这些页面，方便后续查阅、使用或二次开发。所有页面均为静态 HTML，可直接在浏览器中打开运行。

## 目录说明

```
.
├── README.md          # 本文件
├── pages/             # 归档的 HTML 页面存放目录
│   ├── page-001/      # 每个页面一个子目录
│   ├── page-002/
│   └── ...
└── assets/            # 共享资源文件（如有）
```

- `pages/`：每个子目录对应一个独立的 HTML 页面，包含其所需的所有资源（CSS、JS、图片等）。
- `assets/`：存放多个页面可能共用的资源，如通用样式、工具脚本等。

## 页面归档说明

- 归档的页面均为作者原创或已获得授权，不涉及侵权内容。
- 每个页面独立成目录，便于单独部署或引用。
- 页面命名采用 `page-序号` 格式，避免与具体内容挂钩。
- 部分页面可能包含示例数据或占位内容，不代表实际使用场景。

## 维护说明

- 本仓库由维护者不定期更新，新增或清理页面。
- 欢迎通过 Issue 提交页面建议或报告问题，但不保证立即处理。
- 不接受直接向仓库提交 Pull Request 添加第三方页面（除非获得明确授权）。
- 所有页面保持最小依赖，尽量不引入外部 CDN 或第三方服务。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下任意目录中的 `index.html` 文件。
3. 也可通过任意静态文件服务器部署整个仓库目录。
